API Reference

Endpoint

Satu endpoint, satu parameter. Kirim URL target, dapat URL aslinya.

Via query string

GET /api/bypass?url=https://linkvertise.com/... 
const url = "https://linkvertise.com/546946/example";
const res = await fetch(`/api/bypass?url=${encodeURIComponent(url)}`);
const data = await res.json();
console.log(data.result);

Via request body

POST /api/bypass application/json 
const res = await fetch("/api/bypass", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ url: "https://linkvertise.com/..." })
});
const data = await res.json();
console.log(data.result);
curl -X POST https://domain.vercel.app/api/bypass \
  -H "Content-Type: application/json" \
  -d '{"url":"https://linkvertise.com/..."}'

Implementasi

Contoh lengkap — tinggal copy, ganti domain, langsung jalan.

// bypass.js
const bypassURL = async (url) => {
  const res = await fetch(
    `https://domain.vercel.app/api/bypass?url=${encodeURIComponent(url)}`
  );
  const data = await res.json();
  if (data.error) throw new Error(data.error);
  return data.result;
};

// Penggunaan
bypassURL("https://linkvertise.com/546946/example")
  .then(result => console.log("Hasil:", result))
  .catch(err => console.error("Gagal:", err.message));
